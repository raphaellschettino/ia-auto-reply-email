# ia-auto-reply-email
Resposta automática de e-mail.


pip install -r requirements.txt
python.exe -m pip install --upgrade pip

PAra rodar: uvicorn main:app --reload ou python -m uvicorn main:app --reload



O agente rodará automaticamente lerá os e-mails e os responderá.

Antes em utils deve criar credentials.json e incluir as informações da api gmail.
